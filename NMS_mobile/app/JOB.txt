Есть клиент-серверная часть приложения на Expo (React)

Необходимо реализовать:
1. На всех устройствах в () скобках перед подписью в колонке Подпись нужно отображать:
Для всех свитчей в () скобаках количество маков на порту, полчаем по SNMP
Для OLT где с label epon0/1, epon 0/2 и т.д - количество зарегистрированных ОНУ в порту, получаем по SNMP
для OLT где label GE0/1, GE0/2 и т.д. TE0/1 и т.д. - количество маков на порту,  получаем по SNMP
для OLT где number к примеру "{97}0/1:1"  - количество маков на порту, получаем по SNMP, {97} - номер ifindex
